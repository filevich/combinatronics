### Combinatronics



#### 1. combinations example

input:

```
Combs([]int{1, 2, 3, 4, 5}, 3)
```

output:

```
[[3 2 1] [4 2 1] [5 2 1] [4 3 1] [5 3 1] [5 4 1] [4 3 2] [5 3 2] [5 4 2] [5 4 3]]
```

---



#### 2. subsets example

input:

```
Subsets([]int{1, 2, 3}, 3)
```

output:

```
[[1] [2] [3] [2 1] [3 1] [3 2] [3 2 1]]
```
